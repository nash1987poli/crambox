What CramBox is

A single, self-contained study app for ZIMSEC O-Level Maths — built to win where everyone else can't.

The strategy behind it

Blue Ocean thinking, the AI approach, and the payment path.

Blue Ocean — the Four Actions grid

Eliminate

  • Dependence on a tutor being present
  • Heavy data usage
  • The 200-page book nobody finishes

Raise

  • Engagement & fun
  • Instant feedback quality
  • A visible exam-readiness signal

Reduce

  • Price → micro-EcoCash
  • Time-to-value (practise in seconds)
  • Scope → only high-frequency topics

Create

  • Works with zero data
  • AI sparring partner
  • Diaspora gifting (relatives abroad pay)

The AI approach: baked in, not live

🤖

1. AI writes

Claude generates exam-style questions + worked solutions, up front.

→
📦

2. Baked in

They ship inside the HTML file. No API key, no server.

→
📴

3. Runs offline

Zero data, zero per-student cost, safe on shared copies.

Live AI would break offline + cheap (the whole edge) and expose your API bill to every shared file. So the core stays baked-in. Live "Challenge Mode" is parked as a future paid upsell only.

Payment path (EcoCash)

A · Tidy Manual Start here

Pay EcoCash → send screenshot on WhatsApp → you reply with a code. Launch this week, zero build.

No fees · no setup

B · Self-Service Form

Pay, then a form auto-emails a code; you spot-check later.

Semi-automated

C · Full Auto (Paynow)

Paynow confirms payment → code auto-delivered. Accepts cards too — unlocks diaspora parents.

Hands-off · small fee
The strategic link: if you chase diaspora parents (who pay by card), Path C isn't "later" — it's the key, because EcoCash-only locks out your highest-value buyers abroad.
